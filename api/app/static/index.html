LangGraph + OpenAI + arXiv

Search papers through a five-agent research desk.

One agent plans the arXiv query. A second agent reviews fit. A third agent reads the paper and writes key bullet points from the full text. A fourth agent audits the whole workflow. A fifth agent synthesizes literature review outlines from Zotero.

Agent 1

Search Agent

Builds the arXiv query and pulls candidate papers.

Agent 2

Review Agent

Checks whether each paper actually fits your topic.

Agent 3

Summary Agent

Reads the paper PDF and writes 5-10 key bullet points.

Agent 4

Sanity Agent

Checks that query, review, and summary outputs are internally consistent.

Agent 5

Outline Agent

Builds a literature review outline from accepted Zotero references.

Search arXiv

Build an arXiv query from your topic, retrieve candidate papers, score their fit, summarize accepted results, and optionally save accepted papers to Zotero.

Upload + Analyze

Read your uploaded PDFs or Zotero RDF entries, compare each reference against the topic above, keep the best fits, and generate bullet summaries for export.

Upload + Organize

Read uploaded PDFs or Zotero RDF entries without topic scoring, generate bullet summaries for every reference, and prepare the full set for Excel export.

Zotero API

Connect to a personal Zotero library with your username and API key, fetch library items, score them against the topic above, export summarized results, and optionally prune duplicates or low-score items.

Literature Outline

Read a Zotero personal library, let a dedicated outline agent synthesize a literature review structure, and export the result as a Word document with inline APA citations.

Uploaded paper organization

Read uploaded PDFs, generate bullet summaries, and prepare everything for export.

Search mode builds an arXiv query. Switch to Upload PDFs to score and summarize local files.
