# Web App

This is the Next.js frontend for `multiagentsecurity.ai`.

## Current Scope

- App Router setup with TypeScript
- Reusable layout shell
- Home, About, Research, Taxonomy, and Article pages
- Mock query modules that can later be replaced with database-backed implementations
- Health route at `/api/health`

## Commands

- `npm run dev`
- `npm run lint`
- `npm run typecheck`
- `npm run build`
